## 常用命令（复制即用）

### 后端首次启动

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -e ".[dev]"
pip install email-validator
alembic revision --autogenerate -m "init"
alembic upgrade head
uvicorn app.main:app --reload --port 8000
```

### 后端日常启动

```powershell
.\.venv\Scripts\Activate.ps1
uvicorn app.main:app --reload --port 8000
```

### 前端

```powershell
npm run dev
```

### 改表后生成迁移
```powershell
alembic revision --autogenerate -m "describe change"
alembic upgrade head
```
