## 已知 bug / 待优化

- 🐞 **解析状态前端不刷新**：提交日记后前端先返回"解析中"，需要手动刷新页面并再次点"解析"才拿到结果。根因是异步解析是 `BackgroundTasks` + 前端无轮询；需要加轮询或 SSE/WebSocket。
- 🐞 备忘录 `update`（"明天买抽纸" → 第二天"买了抽纸"应勾掉）的端到端需要补一条 e2e 测试。
- 🔧 `LLM_CACHE_ENABLED=false` 默认关（本地无 Redis）；装 Redis 后打开。
- 🔧 日记解析 prompt 当前偶发返回 `data.extra` 里塞主字段——schema 层已兜底，但需要 prompt_fragment 里再强调一次。
